# User Story Map - Techdome Website

## Persona 1: Visitor

### Navigation

US-01:
As a visitor,
I want to navigate through the navbar,
So that I can explore different sections of the website.

US-02:
As a visitor,
I want to access the About Us page,
So that I can learn more about the company.

US-03:
As a visitor,
I want to access the Careers page,
So that I can explore job opportunities.

---

## Persona 2: Potential Customer

### Contact Form

US-04:
As a potential customer,
I want to submit an enquiry through the Contact Us form,
So that I can communicate with the company.

US-05:
As a potential customer,
I want to see validation messages for mandatory fields,
So that I can provide complete information.

US-06:
As a potential customer,
I want to receive a success confirmation after form submission,
So that I know my enquiry was submitted successfully.

US-07:
As a potential customer,
I want the system to validate my email address,
So that I can avoid entering invalid email addresses.

---

## Persona 3: Knowledge Seeker

### Blogs & Articles

US-08:
As a visitor,
I want to read blogs and articles,
So that I can improve my knowledge.

US-09:
As a visitor,
I want to navigate to detailed blog pages,
So that I can read complete content.

---

## Persona 4: Mobile User

### Mobile Experience

US-10:
As a mobile user,
I want the website to be responsive,
So that I can use it comfortably on mobile devices.

US-11:
As a mobile user,
I want all navigation elements to work correctly,
So that I can browse the website without issues.

---

## Persona 5: Subscriber

### Newsletter

US-12:
As a visitor,
I want to subscribe to newsletters,
So that I can receive updates from Techdome.

---

## Persona 6: Social Media User

### Footer

US-13:
As a visitor,
I want to access social media links,
So that I can follow Techdome on different platforms.

---

## Persona 7: Security Conscious User

### Security

US-14:
As a user,
I want the application to validate malicious input,
So that my data remains secure.

US-15:
As a user,
I want the application to implement secure HTTP headers,
So that my browsing experience is secure.
